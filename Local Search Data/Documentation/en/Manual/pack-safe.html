    Verified packages

    Verified packages are packages that have undergone testing with a specific version of Unity, and all other packages verified for that version. Packages that have been verified to work with this version of Unity display the indicator beside it in the Package Manager window. For more information about the verified state and the lifecycle of packages, see Package states and lifecycle.

    The packages that appear in this list are safe to use.
